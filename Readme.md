# Insurance Data Preprocessing and EDA

## Project Overview

This project focuses on Exploratory Data Analysis (EDA), data cleaning, preprocessing, feature engineering, and feature selection using the Insurance Dataset.

The goal is to prepare high-quality structured data before applying Machine Learning models.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SciPy

## Key Tasks Performed

- Dataset loading and inspection
- Null value checking
- Duplicate removal
- Distribution analysis using histograms and boxplots
- Correlation analysis using heatmap
- Label Encoding for binary categorical variables
- One-Hot Encoding for multi-category variables
- BMI Category feature creation
- StandardScaler for feature scaling
- Pearson Correlation for numerical feature selection
- Chi-Square Test for categorical feature selection
- Final dataset preparation for ML model training

## Dataset

Dataset used: Insurance Dataset (`insurance.csv`)

## Final Output

Prepared clean and optimized dataset ready for future ML model training.
